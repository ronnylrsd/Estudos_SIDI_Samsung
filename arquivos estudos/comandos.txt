docker stop (nome_do_container) -> para o container

docker rm (nome_do_container) -> remove o container

docker rmi (nome_da_imagem) -> remove a imagem

docker ps -> lista os containers em execução

docker ps -a -> lista todos os containers

docker images -> lista as imagens

docker run -d -p 8080:80 nginx -> cria um container com a imagem nginx e mapeia a porta 8080 para a porta 80 do container rodando em background.

docker run -it node -> cria um container com a imagem node e abre um terminal interativo

docker start (nome_do_container) -> inicia um container fechado ou parado

docker logs (nome_do_container) -> mostra os logs do container
